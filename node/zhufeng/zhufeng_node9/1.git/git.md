## 在github上部署静态页
需要将特定的内容推送到github上的gh-pages分支上

- 创建一个gh-pages的分支
```
git checkout -b gh-pages
```
- 将代码commit 到这个分支上
```
git add .
git commit -m 'ok'
```
- 连接远程仓库和本地仓库
```
git remote add aaa 地址
```
- 将代码推送到远程仓库上
```
git push aaa gh-pages
```
